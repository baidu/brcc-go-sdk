# 简介
&ensp;&ensp;brcc-go-sdk是配置中心BRCC(https://github.com/baidu/brcc)的golang客户端。BRCC是一个分布式配置中心，用于统一管理应用服务的配置信息，避免各类资源散落在各个项目中，简化资源配置的维护成本。作为一种轻量级的解决方案，部署简单，同时支持多环境、多版本、多角色的资源管理，可以在不改变应用源码的情况下无缝切换和实时生效配置信息。
